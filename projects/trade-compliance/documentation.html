PROJECT: GLOBAL TRADE ANALYSIS ENGINE • FILE ID: KD-GTA-002-DOC

SYSTEM OVERVIEW

The Global Trade Analysis Engine is a client-side JavaScript application designed to provide high-level estimates for landed costs in international trade. It utilizes a static dataset representing the top 10 global economies and core product categories. The engine models complexities such as Free Trade Agreements (FTAs) and differential logistics costs for sea and air freight.

The primary goal of this project is to demonstrate data modeling and interactive application development skills, not to provide financially accurate quotes for real-world shipping.

DATA MODEL & LOGIC

Dataset

The application's logic is powered by a static JavaScript object containing data for 10 countries. The key data points are:

  • Country List: ISO codes and full names for the top 10 global economies.
  • Tax Rates: A dictionary mapping each country ID to a representative Value-Added Tax (VAT) or Goods and Services Tax (GST) rate.
  • Base Duty Rates: A baseline import duty percentage for each product category.
  • Trade Agreements (FTAs): A dictionary that defines duty multipliers for specific trade routes (e.g., a value of `0` for EU internal trade, `0.5` for a 50% duty reduction).

Calculation Flow

The calculation logic proceeds as follows:

  1. Determine the base duty rate from the selected product category.
  2. Check if a Free Trade Agreement exists between the origin and destination. If so, apply the FTA multiplier to the base duty rate.
  3. Calculate shipping cost based on the selected mode (Air/Sea) and the shipment value.
  4. Calculate import duty based on the final duty rate and shipment value.
  5. Calculate the taxable amount (Shipment Value + Logistics Costs + Duty).
  6. Calculate the final tax based on the destination country's tax rate.
  7. Sum all components to produce the Total Landed Cost.

CLIENT-SIDE APPLICATION (app.js)

Core Class: GlobalTradeModel

All calculations are encapsulated within this class. It is initialized with all user inputs from the dashboard.

const model = new GlobalTradeModel(origin, destination, category, value, insurance, shippingMode);
const results = model.calculate();

PROJECT: GLOBAL TRADE ANALYSIS ENGINE (KD-GTA-002)

SYSTEM OVERVIEW

The Global Trade Analysis Engine is a client-side JavaScript application designed to provide high-level estimates for landed costs in international trade. It utilizes a static dataset representing simplified duty rates and trade agreements.

DATA MODEL & LOGIC

The application's logic is powered by a static JavaScript object in app.js. Key data points include Base Duty Rates, Free Trade Agreements (FTAs), and currency conversion rates.

CALCULATION FLOW

  1. Determine the base duty rate from the selected product category.
  2. Check if an FTA exists to potentially waive the duty.
  3. Calculate final landed cost including shipping and insurance estimates.

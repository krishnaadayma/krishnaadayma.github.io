PROJECT: GLOBAL TRADE ANALYSIS ENGINE (KD-GTA-002)

SYSTEM OVERVIEW

The Global Trade Analysis Engine is a client-side JavaScript application designed to provide high-level estimates for landed costs in international trade. It utilizes a static dataset representing macroeconomic indicators and trade policies for the world's top 10 economies.

DATA MODEL & LOGIC

The application's logic is powered by a static JavaScript object in app.js. The key data points are:

  • Base Duty Rates: Default import tariffs per product category.
  • Free Trade Agreements (FTAs): A matrix defining tariff exemptions between countries.
  • Shipping Costs: Estimated costs per mode (Sea/Air) between country pairs.

CALCULATION FLOW

  1. Determine the base duty rate from the selected product category.
  2. Check if a Free Trade Agreement exists between the origin and destination to potentially waive the duty.
  3. Calculate the final duty and VAT based on the shipment value.
  4. Add estimated shipping and insurance costs to arrive at the Total Landed Cost.

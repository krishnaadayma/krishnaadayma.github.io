PROJECT: GLOBAL TRADE ANALYSIS ENGINE (KD-GTA-002)

SYSTEM OVERVIEW

The Global Trade Analysis Engine is a client-side JavaScript application designed to provide high-level estimates for landed costs in international trade. It utilizes a static dataset representing macroeconomic indicators and trade policies for the world's top 10 economies.

DATA MODEL & LOGIC

The application's logic is powered by a static JavaScript object in app.js. Key data points include Base Duty Rates, Free Trade Agreements (FTAs), and Shipping Costs.

CALCULATION FLOW

  1. Determine the base duty rate from the selected product category.
  2. Check if an FTA exists to potentially waive the duty.
  3. Calculate final duty and VAT.
  4. Add shipping and insurance to arrive at the Total Landed Cost.

📋 System Overview

The Global Trade Analysis Engine is a client-side JavaScript application designed to provide high-level estimates for landed costs in international trade. It utilizes a static dataset representing simplified duty rates and trade agreements.

Client-Side Vanilla JS Static Data Educational

🗄️ Data Model & Logic

The application's logic is powered by a static JavaScript object. Key data points include:

  • Base Duty Rates — Per product category (Machinery: 5.5%, Textiles: 12%, etc.)
  • Country-Specific Rates — Override base rates for specific jurisdictions
  • Free Trade Agreements — Bilateral and multilateral treaty database
  • Currency Conversion — Static rates for cost normalization
  • Shipping Rates — Modal cost factors (Sea: 3%, Air: 12%, etc.)
// Duty rate structure const DUTY_RATES = { machinery: { base: 5.5, IT: 0, IN: 7.5, US: 2.5, ... }, textiles: { base: 12.0, IT: 0, IN: 15, US: 8.0, ... }, // ... additional categories };

  1. Determine the base duty rate from the selected product category
  2. Check if an FTA exists to potentially waive the duty
  3. Calculate CIF (Cost + Insurance + Freight)
  4. Apply duty to CIF value
  5. Add destination VAT/GST on (CIF + Duty)
  6. Sum all components for total landed cost

⚡ API Reference

calculate(origin, destination, category, value, insurance, mode)

Main calculation function. Returns an object with all cost components.

// Example usage const result = calculate({ origin: 'IT', destination: 'IN', category: 'machinery', value: 50000, insurance: 1.5, mode: 'sea' }); // Returns: // { cif: 51750, duty: 3881.25, vat: 10013.63, total: 65644.88 }

🤝 FTA Logic

The engine checks for active Free Trade Agreements between origin and destination countries. If an active FTA is found, the duty rate is set to 0.

const FTAS = [ { from: 'IT', to: 'DE', name: 'EU Single Market', active: true }, { from: 'IN', to: 'AE', name: 'India-UAE CEPA', active: true }, { from: 'IT', to: 'IN', name: 'EU-India FTA', active: false }, // ... additional agreements ];

Pending FTAs are displayed as informational badges but do not affect duty calculations.
